# Contributing to Agentic CI/CD Security

Thank you for your interest in contributing.

Agentic CI/CD Security is an open-source knowledge base for securing AI agents in CI/CD pipelines and software delivery workflows.

## Ways to Contribute

You can contribute by:

- Adding security checklist items
- Improving threat models
- Adding vulnerable workflow examples
- Adding hardened workflow examples
- Adding research papers
- Adding defensive tools
- Adding case studies
- Fixing unclear explanations
- Correcting unsafe or misleading guidance

## Contribution Guidelines

Please keep contributions:

- Relevant to AI agents, CI/CD, DevSecOps, or software supply-chain security
- Practical and security-focused
- Clearly written
- Vendor-neutral where possible
- Supported by references where appropriate
