# Security Policy

## Reporting Security Issues

This repository contains security guidance, examples, and educational material.

If you find unsafe guidance, misleading examples, incorrect assumptions, or security issues in this project’s recommendations, please open an issue or submit a pull request with a correction.

## What to Report

Please report:

- Unsafe recommendations
- Misleading examples
- Incorrect security assumptions
- Broken links to important references
- Workflow examples that may cause unintended harm
- Missing warnings around risky patterns
- Overly broad or unclear security advice

## Example Safety

Vulnerable examples in this repository are for educational and defensive purposes only.

They should not be copied into production environments.

## Responsible Use

Do not use this project to attack systems, steal secrets, bypass CI/CD protections, or compromise software delivery pipelines.

The goal of this project is defensive security education and safer AI-agent adoption.
